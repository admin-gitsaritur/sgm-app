### 3. Correções de UI/UX

#### 3.1 Tooltip Padrão
- Substituído `title="Alterar foto"` (HTML nativo) pelo componente `<Tooltip>` do Radix UI com `TooltipProvider`, `TooltipTrigger`, e `TooltipContent`.

#### 3.2 Botão Câmera — Efeito Vidro (Glassmorphism)
- Fundo: `bg-white/70` com `backdrop-blur-md`
- Hover: `bg-white/90` + elevation `shadow-lg`
- Ícone: cinza por padrão, vermelho no hover

#### 3.3 Modal — Fundo Branco + Desfoque
- `DialogOverlay`: `bg-black/40` + `backdrop-blur-sm` (antes: `bg-black/50` sem blur)
- `DialogContent`: `bg-white` (antes: `bg-background` — resolvia para cinza em alguns contextos)

#### 3.4 Modal Crop — Footer Fixo
- Botões "Cancelar" e "Confirmar" movidos para a prop `footer` do `<FormModal>`
- Footer fica fixo na base do modal, independente do tamanho da imagem

#### 3.5 Modal Alterar Senha — Footer Fixo
- Mesma correção do modal de crop: botões via prop `footer` + atributo `form="password-form"` para submit funcionar

### 5. Mapeamento Estratégico (`/mapeamento`)

- **Loader padrão:** Spinner CSS manual substituído pelo componente `<Spinner>` do UI Kit
